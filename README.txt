Репозиторий проекта "Ычан-Эроге" от 2013 года. Каждый новый вариант правки - отдельный форк.
Даже одна небольшая правка становится потенциальным вкладом в дальнейшую разработку.
Также для исторической справки и референсов предоставлены сценарные файлы демо-версии 2009 года, версии, выложенные Сорком (Sorc_ver, с наработками по 3-5 дням).
Все материалы по разработке от 2012 года находятся в master-ветке eroge-dopil.
Диздок ранней версии находится в соответствующей папке.
